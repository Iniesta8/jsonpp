jsonpp by jmhodges

A fast command line JSON pretty printer.

jsonpp is a simple to install and fast to run command line JSON pretty printer. You'll love it.

Install

Grabbing the binary is the easiest way to do install jsonpp. It's a simple file.

To begin, you'll want to download the zip file that matches your machine:

  • jsonpp for OS X
  • jsonpp for Linux

Then, simply decompress those files and copy the jsonpp binary to somewhere in your $PATH. All you have left to do is to run:

curl -s -L http://t.co/tYTq5Pu | jsonpp

The source install requires a working Go install. Once it's installed run goinstall github.com/jmhodges/jsonpp, or run make install yourself.

Dependencies

Nada.

Unless you build from source, and then it depends on a Go install.
